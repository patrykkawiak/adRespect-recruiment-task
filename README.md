## Run Locally

Clone the project

```bash
  git clone https://github.com/patrykkawiak/adRespect-recruiment-task.git
```

Go to the project directory

```bash
  cd adRespect-recruiment-task
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

The server will start at: http://localhost:3000/
